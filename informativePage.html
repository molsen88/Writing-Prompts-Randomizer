Informative Writing

“It is a very sad thing that nowadays there is so little useless information.” ― Oscar Wilde

Writing with Text Structure

When writing an informative essay, it is important to remember your purpose in writing an informative paper and text structure. An author should have a specific purpose for writing an informative paper. You not only want to inform a reader about a particular topic, but how you organize your paper depends on your purpose.

Text structure is how an author organizes the text. Once an author determines their purpose, they will choose the text structure they will need to effectively communicate and organize their paper. There are five different text structures an author uses when writing an informative essay:

  • Description Learn more!

    • Description Text Structure

    Purpose:

    To describe a person, place, thing, or idea.

    Signal Words:

    for example, including, is, is like, can be described as, following, another, labeled as, known as, defined as

    Organize:

    An author chooses a main idea to convey to the reader about a given topic. A good description text structure includes like information grouped together in sections. Each section,depending on the length, should have a sub-heading that addresses a more specific topic that relates to the main idea of the text.

    Close
  • Compare and Contrast Learn more!

    • Compare and Contrast Text Structure

    Purpose:

    To write about similarities and differences between two topics, people, places, things, or ideas.

    Signal Words:

    similar, alike, compared to, both, similarity, correlates,resembles, relates to, difference, contrasting, unlike, differs from, opposite, however, while, not related to

    Organize:

    An author might choose to organize the information either by the two topics or by grouping the similarities and the differencs in to two sections.

    Close
  • Sequence Learn more!

    • Sequence

    Purpose:

    To write about a person, place, thing, events, or ideas in time order or in an order that needs to be followed.

    Signal Words:

    first, second, third, beginning, after, next, later, lastly, another, dates references, time references, on, following, then, when, before

    Organize:

    An author would use sequence text structure to describe how a particular event took place, how make something in steps, or to write about someone's life. Writing in sequence text structure requires information to be grouped in the order it happened or needs to be done.

    Close
  • Cause and Effect Learn more!

    • Cause and Effect Text Structure

    Purpose:

    An author writes about an event or several events that lead to another event or a series of events. The author shows the connection between an action and the result or results of the action(s)

    Signal Words:

    cause, effect, result, as a result, caused by, because, so, consequently, therefore, since, thus, leads to, as a consequence of, due to, so that

    Organize:

    An author would use cause and effect to show how events are related. When one event leads to another or when one event causes another event to happen. An author should organize the text by connecting events together or in sections. An author may also choose to group the events in a section and follow it up with another section of the consequences or the results that happen or could happen.

    Close
  • Problem and Solution Learn more!

    • Problem and Solution Text Structure

    Purpose:

    An author presents the reader with a problem and then lists a solution or possible solutions to the problem. A reader should be able to identify what the problem is and then describe the various solutions an author presents.

    Signal Words:

    problem, solution, issue, answer, possibility, resolution(s), if...then, because, this led to, cause, could be, accordingly

    Organize:

    An author will first present a problem(s) that needs to be addressed. The author should write about why the problem needs to be addressed. After the problem is presented, an author should write about a solution or possible solutions to the problem that was presented. An author may organize the information by presenting the problem and then follow it with solutions. Separate sections for the problem and solution may be helpful, but at the least, the problem should be presented first followed by possible solutions.

    Close
